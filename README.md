# Gerenciador Escolar Financeiro 🎒

  Um projeto escolar desenvolvido com o objetivo de gerenciar os patrimônios, por exemplo cadeiras, mesas e eletrônicos de uma instituição escolar.

- HTML
- CSS
- Java Script
- Flask
- Flask_login
- SQLite
